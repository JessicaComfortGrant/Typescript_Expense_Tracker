# TypeScript Expense Tracker

A simple expense tracking application built with **TypeScript**.

This project was created to practice core TypeScript concepts including interfaces, classes, type safety, modules, and basic data management.

## Features

- Add expenses
- View all expenses
- Calculate total spending
- Organize expenses by category
- Track expense dates

## Technologies Used

- TypeScript
- Node.js
- npm

## Installation

Clone this repository:

```bash
git clone https://github.com/YOUR_USERNAME/typescript-expense-tracker.git
```

Navigate into the project:

```bash
cd typescript-expense-tracker
```

Install dependencies:

```bash
npm install
```

Run the project:

```bash
npx tsc
node src/index.js
```
